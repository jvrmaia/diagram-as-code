Pagar.me Infrastructure & Applicatios diagram
----

# Requirements

* Python 3.8^
* Poetry (pip install poetry==1.0.5)
* Graphviz 2.42.3

# Setup

Install dependencies:

> Disclaimer: it's recommend setup a virtualenv to the project

```
poetry install
```

# Generate diagram

```
cd <project>
python <project>.py
```
